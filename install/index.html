Installation

You can use the following methods to install kaldifst on Windows, Linux, and macOS. Python >= 3.6 is known to work.

Using conda

conda install -c kaldifst kaldifst

To uninstall kaldifst, use:

conda remove kaldifst

Using pip

pip install --verbose kaldifst

or

pip install --verbose -U git+https://github.com/kaldifst/kaldifst.git

To uninstall kaldifst, use:

pip uninstall kaldifst

From source

git clone https://github.com/csukuangfj/kaldifst
cd kaldifst
python3 setup.py install

To uninstall kaldifst, use:

pip uninstall kaldifst

Check kaldifst

To check that kaldifst was installed successfully, use:

python3 -c "import kaldifst; print(kaldifst.__version__)"

It should print the version of kaldifst, e.g., 0.8.
